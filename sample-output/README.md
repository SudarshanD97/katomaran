# Sample Output

This folder contains example artifacts that show how the backend should persist results.

Files included:

- `events.log` for critical system events.
- `visitor_events.json` for sample database rows.

Expected image layout:

```text
logs/
  entries/YYYY-MM-DD/
  exits/YYYY-MM-DD/
```

The repository also includes synthetic example images in `public/sample-output/` for presentation purposes.
